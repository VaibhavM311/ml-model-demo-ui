# 📊 AI Resume Screening SaaS Dashboard

An enterprise-style ATS dashboard for AI-powered candidate evaluation.

## 🚀 Features
- Real-time hiring decision
- Candidate score meter
- Corporate ATS dashboard UI
- Quick example candidate testing
- Machine learning powered prediction
- Gradio web interface
- Hugging Face deployment ready

## 🛠️ Tech Stack
- Python
- Gradio
- Scikit-learn
- NumPy
- Pandas

## 📌 Use Case
This project helps recruiters and HR teams quickly evaluate candidate resumes using machine learning and provides an ATS-style hiring dashboard.
